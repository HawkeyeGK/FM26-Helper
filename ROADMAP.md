# FM26-Helper Roadmap

## Phase 3: Enhanced UI & Interaction
- [ ] **Sorting & Filtering:** Add ability to sort the Roster Grid by any column (Speed, Role Score, Wage).
- [ ] **Player Details View:** Click a row to see the full "Player Card" (Raw Attributes + Analysis).
- [ ] **Visual Polish:** Add color-coding (Heatmap style) for high attributes (16+) and high role scores (80%+).

## Phase 4: Data Management
- [ ] **Edit Player:** Ability to fix OCR errors via the Web UI.
- [ ] **Add/Remove Player:** UI controls to delete players or manually add one.
- [ ] **Goalkeeper Support:** Update Data Models to handle GK attributes (Reflexes, Handling) correctly.

## Phase 5: Advanced Analytics
- [ ] **Comparison Tool:** Select two players and view a "Head-to-Head" chart.
- [ ] **Squad Planner:** Visualizing depth charts by position.
- [ ] **Evolution Tracking:** Storing history of attributes to show development over time.
